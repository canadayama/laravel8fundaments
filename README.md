# Laravel 8 Fundamentals

## Create Laravel Project

`$ composer create-project laravel/laravel:8.* project-name`

### Laravel Version

`$ php artisan --version`

## Environment Variables

In source:

`env('ENVIRONMENT_VARIABLE');`

In .env:

`ENVIRONMENT_VARIABLE=Value`

## Composer

`$ composer install`

`$ composer update`

`$ composer show --tree`

`$ composer dump-autoload`
